# editable-csv-viewer

A tool that allows you to drop in a .csv file and that displays all the data in an editable table. I want to also be able to add and remove rows and do download the edited version of the csv file.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/editable-csv-viewer.git
cd editable-csv-viewer
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
